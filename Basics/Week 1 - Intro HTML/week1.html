I have included the first module I created that allows me to quickly Set the Mailbox Quota of a users Exchange Mailbox.

The Input takes a Username and the Quota you want to set in GigaBites.
The Output is a custom object with fields that let you know the respective values (Note: these values are not formatted by design).
